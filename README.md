# Potion Shop
A simple and elegant online store for magical potions. Browse the catalog, place orders via a form, and view your order history.

# Installation & Development
## Clone the repository
git clone https://github.com/KiriaLbk111/ShopProject.git
## Install dependencies
npm install
## Start development server
ng serve

# Running Tests
## Run unit tests once
ng test --no-watch --no-progress

## Run tests in watch mode (with browser)
ng test
